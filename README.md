# 📄 Interactive CV Builder

Kullanıcıların kendi profesyonel özgeçmişlerini anlık olarak tasarlayabilecekleri, dinamik ve kullanıcı dostu bir web uygulaması.

## ✨ Özellikler
- **Anlık Önizleme:** Form alanlarına girilen bilgiler anında CV üzerinde güncellenir.
- **Görünüm Özelleştirme:** Renk paleti, yazı tipi ve stil seçimleri.
- **Client-side:** Tamamen tarayıcı üzerinde çalışır, kurulum gerektirmez.

## 🚀 Teknolojiler
- HTML5, CSS3,  JavaScript

## 🛠️ Kurulum
1. Klasörü indirin.
2. \`index.html\` dosyasını tarayıcıda açın.
